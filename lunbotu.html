抽牌效果之轮播图

banner轮播方式真的很多，这也算一种特别的轮播方式了吧~

css3结合jq，炫的都是css3的功劳。

CSS:
                
               /*图片轮播*/
               #frame{width:1000px;height:400px;overflow:hidden;border-radius:5px}
               #photos img{float:left;width:1000px;height:400px}
               #photos {z-index:9;  width: calc(1000px * 3);/*---修改图片数量的话需要修改下面的动画参数*/  }
               .play{ animation: ma 5s ease-out infinite alternate;}
               @keyframes ma {
               0%,30% {        margin-left: 0px;       }         
               35%,65% {       margin-left: -1000px;    }
               70%,100% {      margin-left: -2000px;    }
               }
